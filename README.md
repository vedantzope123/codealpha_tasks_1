# Calculator Pro

Advanced calculator app with modern glassmorphism design.

## Features
- All arithmetic operations (+, -, ×, ÷)
- Real-time result display
- Operation history
- Keyword support (add, sqrt, power, pi, etc.)
- Keyboard support
- Responsive design
- Glassmorphism UI

## Usage
1. Click buttons or use keyboard
2. Type keywords like "sqrt", "power", "pi"
3. View history and click to reuse
